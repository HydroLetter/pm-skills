# 产品经理资源工具箱

本skill提供PM职业成长全路径、工具资源清单、学习资料汇总。

## 一、PM成长路线

### 标准PM能力进阶路线（0→1）

```
阶段0：基础认知
  ├→ 了解产品经理是什么、做什么
  ├→ 明确自己是否适合这个职业
  └→ 建立产品思维基础

阶段1：能力搭建
  ├→ 掌握产品工具（Axure/Xmind/ProcessOn）
  ├→ 学习需求分析方法
  ├→ 能独立输出PRD和原型
  └→ 了解产品开发全流程

阶段2：实战积累
  ├→ 参与完整项目周期
  ├→ 独立完成需求从0到1
  ├→ 积累行业认知和用户洞察
  └→ 建立跨部门协作能力

阶段3：专业深化
  ├→ 形成方法论体系
  ├→ 能带领小团队
  ├→ 具备商业敏感度
  └→ 有成功项目案例

阶段4：专家/管理
  ├→ 产品线负责人
  ├→ 制定产品战略
  ├→ 培养新人
  └→ 推动业务增长
```

### AI产品经理专项学习路线

| 阶段 | 时长 | 学习内容 | 产出目标 |
|------|------|----------|----------|
| **全局了解** | 1天 | AI产品架构全景、岗位分析、个人规划 | 明确AI PM定位 |
| **Python编程** | 10天 | 基础语法、面向对象、爬虫实战 | 能读懂代码、写简单脚本 |
| **机器学习** | 15天 | 基础概念、算法类型、常用算法、案例分析 | 理解ML原理和流程 |
| **深度学习** | 12天 | 神经网络、CV、NLP、GAN | 了解深度学习应用场景 |
| **AI产品设计** | 20天 | 竞品调研、PRD、模型训练、工程化 | 能独立设计AI产品 |
| **AI项目管理** | 10天 | 敏捷/瀑布管理、进度管理、工具 | 能管理AI项目交付 |
| **简历面试** | 2天 | 岗位定位、简历修改、面试技巧 | 拿到AI PM offer |

**总计约70天**（全职投入），兼职建议3-6个月。

## 七、AI产品经理专项

### AI知识体系架构

```
人工智能
  ├→ 基础层
  │   ├→ Python编程
  │   ├→ 数学基础（线性代数、概率论、微积分）
  │   └→ 数据处理（Pandas/Numpy）
  │
  ├→ 算法层
  │   ├→ 机器学习（监督/无监督/强化）
  │   ├→ 深度学习（CNN/RNN/Transformer）
  │   └→ 大模型（GPT/LLM）
  │
  ├→ 应用层
  │   ├→ 计算机视觉（CV）
  │   ├→ 自然语言处理（NLP）
  │   ├→ 语音识别（ASR）
  │   └→ 推荐系统
  │
  └→ 产品层
      ├→ AI产品调研
      ├→ 模型评估指标
      ├→ 数据标注管理
      └→ 工程化部署
```

### AI PM vs 普通PM差异

| 维度 | 普通PM | AI PM |
|------|--------|-------|
| 需求确定性 | 相对明确 | 不确定性高 |
| 评估指标 | 功能可用性 | 模型准确率/召回率 |
| 迭代周期 | 周/月 | 模型训练周期较长 |
| 协作角色 | 设计/开发/测试 | +算法/数据标注 |
| 验收方式 | 功能验收 | 离线指标+在线A/B |

## 八、PM简历与面试

### 简历撰写要点

1. **STAR法则**：情境→任务→行动→结果
2. **数据量化**：不要只说"提升了用户体验"，要说"DAU提升30%"
3. **突出成果**：重点写产品成果，而非流程描述
4. **项目选择**：选最有代表性的2-3个项目深度写

### 面试常见问题

| 类型 | 问题示例 | 考察点 |
|------|----------|--------|
| **产品思维** | 分析一款你喜欢的产品 | 产品sense |
| **需求分析** | 如何做需求优先级排序 | 方法论 |
| **项目深挖** | 这个项目最大的挑战是什么 | 实战能力 |
| **数据分析** | 怎么看这个功能的效果 | 数据思维 |
| **情景题** | 如何提升某产品的留存 | 解决能力 |
| **开放题** | 估算北京有多少个加油站 | 逻辑思维 |

### 面试准备清单

- [ ] 准备3个完整项目案例（STAR格式）
- [ ] 体验目标公司的产品，准备优化建议
- [ ] 了解行业趋势和竞品动态
- [ ] 准备反问面试官的问题
- [ ] 练习自我介绍（1分钟/3分钟版本）
