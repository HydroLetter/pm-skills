# 产品运营实战指南

本skill提供产品运营全链路实操指南，覆盖指标体系、增长策略、用户运营、渠道运营。

## 一、APP运营指标体系

### 核心指标金字塔

```
         北极星指标
       （产品核心价值）
              ↓
    增长指标    活跃指标    营收指标
   （AARRR）  （DAU/MAU）  （ARPU/GMV）
              ↓
    细分指标（功能/内容/渠道）
```

### 1. 用户活跃指标

| 指标 | 定义 | 计算方式 | 重要性 |
|------|------|----------|--------|
| **DAU** | 日活跃用户 | 当日登录用户数 | ★★★★★ |
| **MAU** | 月活跃用户 | 当月登录用户数 | ★★★★★ |
| **DAU/MAU** | 用户粘性 | DAU÷MAU | 反映用户粘性 |
| **活跃度** | 使用深度 | 使用时长×频次 | 产品健康度 |

**DAU细分指标**：
- 新增用户（当日首次登录）
- 回流用户（7天未登录后回归）
- 活跃用户（连续活跃）

### 2. 用户留存指标

| 留存类型 | 定义 | 健康值参考 |
|----------|------|------------|
| **次日留存** | 注册后次日登录 | 40%+（优秀） |
| **3日留存** | 注册后第3天登录 | 30%+ |
| **7日留存** | 注册后第7天登录 | 20%+ |
| **30日留存** | 注册后第30天登录 | 10%+ |

**留存计算公式**：
> N日留存率 = 第N日仍登录的用户数 ÷ 首日新增用户数 × 100%

### 3. 用户获取指标

| 指标 | 定义 | 优化方向 |
|------|------|----------|
| **新增用户** | 首次使用产品的用户 | 渠道优化 |
| **获客成本(CAC)** | 获取单个用户成本 | 效率提升 |
| **渠道转化率** | 从曝光到注册的比例 | 漏斗优化 |
| **病毒系数(K)** | 每个用户带来新用户数 | 裂变机制 |

### 4. 营收指标

| 指标 | 定义 | 适用场景 |
|------|------|----------|
| **ARPU** | 每用户平均收入 | 评估变现效率 |
| **ARPPU** | 每付费用户平均收入 | 评估付费深度 |
| **LTV** | 用户生命周期价值 | 长期价值评估 |
| **GMV** | 成交总额 | 电商核心指标 |
| **付费率** | 付费用户÷总用户 | 转化能力 |

**LTV计算**：
> LTV = ARPU × 平均用户生命周期(月)

**CAC:LTV健康比例**：
> LTV ÷ CAC ≥ 3:1（可持续增长）

## 二、用户增长方法论

### 增长黑客增长引擎

```
增长本源
    ├→ 转化率的提升
    │       ├→ 识别（问题）
    │       └→ 增长（业务）
    │
    └→ 数据洞察
            ├→ 精细化运营
            └→ 转化率提升
```

### 用户增长实验流程

| 步骤 | 动作 | 产出 |
|------|------|------|
| 1. 提出想法 | 基于数据洞察提出假设 | 想法清单 |
| 2. 排序优先级 | ICE评分(Impact/Confidence/Ease) | 优先级列表 |
| 3. 执行测试 | 快速上线实验版本 | 实验数据 |
| 4. 实验复盘 | 分析结果，总结规律 | 复盘中档 |

### 渠道增长的5个关键节点

```
流量获取 → 新用户 → 活跃用户 → 忠诚用户 → 流失用户
    ↓          ↓         ↓          ↓          ↓
  获客成本    激活率     留存率     付费率     召回率
```

**渠道投放评估维度**：
1. **成本大小**：CAC是否可控
2. **投放精度**：人群匹配度
3. **可控性**：能否快速调整
4. **时间成本**：见效周期
5. **产出周期**：持续产出能力

### 用户自传播增长

**裂变增长关键步骤**：

```
1. 设计分享动机（利益点/情感点）
        ↓
2. 简化分享路径（一键分享/自动生成）
        ↓
3. 接收方激励（新用户福利）
        ↓
4. 数据追踪（归因链路）
        ↓
5. 持续优化（A/B测试）
```

**常见裂变玩法**：
| 玩法 | 核心逻辑 | 案例 |
|------|----------|------|
| **拼团** | 多人低价购买 | 拼多多 |
| **砍价** | 好友助力降价 | 每日一淘 |
| **助力** | 好友帮忙解锁 | 朋友圈打卡 |
| **分销** | 分享得佣金 | 知识星球 |
| **邀请码** | 邀请奖励双方 | Uber |

## 三、用户生命周期运营

### 用户生命周期分层

```
新用户 → 成长用户 → 成熟用户 → 衰退用户 → 流失用户
   ↓         ↓          ↓          ↓          ↓
 激活      留存        变现        预警        召回
```

### 各阶段运营策略

| 阶段 | 核心目标 | 运营策略 |
|------|----------|----------|
| **新用户** | 快速激活 | 新手引导、新人福利、Onboarding |
| **成长用户** | 习惯养成 | 任务体系、签到、等级成长 |
| **成熟用户** | 价值挖掘 | 会员体系、增值服务、个性化推荐 |
| **衰退用户** | 预警干预 | 推送触达、优惠券、专属活动 |
| **流失用户** | 召回激活 | 短信/邮件、回归礼包、问卷调查 |

### 用户分群运营(RFM模型)

| 维度 | 含义 | 分群策略 |
|------|------|----------|
| **R-Recency** | 最近消费时间 | 近期活跃→保持，长期未消费→召回 |
| **F-Frequency** | 消费频率 | 高频用户→VIP，低频用户→促活 |
| **M-Monetary** | 消费金额 | 高价值→专属服务，低价值→提频 |

**8大用户分群**：
- 重要价值用户（高R高F高M）→ 核心维护
- 重要保持用户（低R高F高M）→ 防流失
- 重要发展用户（高R低F高M）→ 提频
- 重要挽留用户（低R低F高M）→ 重点召回
- 一般价值用户（高R高F低M）→ 提客单
- 一般保持用户（低R高F低M）→ 观察
- 一般发展用户（高R低F低M）→ 培养
- 一般挽留用户（低R低F低M）→ 放弃

## 四、渠道运营

### 渠道分类与特征

| 渠道类型 | 特点 | 适合阶段 |
|----------|------|----------|
| **应用商店** | 流量大、质量参差 | 所有阶段 |
| **广告投放** | 可控、可规模化 | 成长期 |
| **内容营销** | 成本低、长尾效应 | 早期/长期 |
| **社群运营** | 精准、互动性强 | 成熟期 |
| **线下推广** | 本地化、成本高 | 特定场景 |
| **异业合作** | 资源互换 | 资源期 |

### 渠道评估指标

| 指标 | 定义 | 健康值 |
|------|------|--------|
| **新增量** | 每日新增用户数 | 持续增长 |
| **留存质量** | 渠道用户7日留存 | ≥平均留存 |
| **付费质量** | 渠道用户付费率 | 符合目标 |
| **ROI** | 投入产出比 | ≥1:3 |

## 五、活动策划

### 活动运营6步法

```
1. 目标设定（拉新/促活/变现/品牌）
    ↓
2. 用户分析（目标人群/需求/场景）
    ↓
3. 方案设计（玩法/奖励/规则/风险）
    ↓
4. 资源准备（预算/流量/人力/物料）
    ↓
5. 上线执行（预热/爆发/收尾）
    ↓
6. 复盘总结（数据/问题/经验）
```

### 常见活动类型

| 活动类型 | 目标 | 适用场景 |
|----------|------|----------|
| **限时折扣** | 促转化 | 清库存/大促 |
| **满减/满赠** | 提客单 | 日常促销 |
| **秒杀** | 引流 | 新品首发 |
| **抽奖** | 促活 | 节日活动 |
| **任务体系** | 养成习惯 | 长期运营 |
| **排行榜** | 制造竞争 | 社区产品 |

## 六、内容运营

### 内容运营闭环

```
内容生产 → 内容审核 → 内容分发 → 内容消费 → 内容互动 → 数据反馈
    ↑                                                   ↓
    └──────────────── 内容优化 ← 用户反馈 ←──────────────┘
```

### 内容分发策略

| 策略 | 逻辑 | 适用 |
|------|------|------|
| **编辑推荐** | 人工精选 | 冷启动/头部内容 |
| **算法推荐** | 个性化 | 成熟期产品 |
| **关注流** | 时间序 | 社交产品 |
| **热门排行** | 热度排序 | 榜单场景 |

```
   是什么？     为什么？     会怎样？     怎么办？
```
